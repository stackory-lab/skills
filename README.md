# Stackory Skills

Consumer-facing Codex/Claude Code skills for using Stackory shared packages.

- `stackory-backend-usage`: Hono middleware, internal auth, service calls, Cloudflare platform adapters.
- `stackory-auth-usage`: PKCE, token storage, device flow, Better Auth client and server integration.
- `stackory-signals-usage`: analytics clients, PostHog adapters, user signals, experiments.
- `stackory-foundation-usage`: storage adapters, shared contracts, standard errors, API and utility helpers.
